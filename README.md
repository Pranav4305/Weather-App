## 📌 About The Project

This is a highly polished, frontend-only single-page application that provides real-time weather information. Built utilizing a modern **Glassmorphism** design aesthetic, it provides a stunning and dynamic user experience.

### ✨ Key Features
- **Dynamic Backgrounds:** Background images automatically change based on the real-time weather conditions of the searched city, utilizing high-resolution imagery.
- **Auto-Location:** Uses browser Geolocation to automatically detect and display the weather for your current city on load.
- **Premium Aesthetics:** Features a beautiful frosted glass UI (`backdrop-filter`), fluid CSS layouts, and modern typography.
- **Micro-Animations:** Includes smooth hover states, dynamic loading skeletons, and a 3D tilt effect on the main weather card.

## 🛠️ Built With

* **HTML5 & CSS3** (Vanilla CSS with CSS Variables for theming)
* **Vanilla JavaScript** (Modern ES6+ with `async/await`)
* **[vanilla-tilt.js](https://micku7zu.github.io/vanilla-tilt.js/)** (For the 3D interactive card effect)
* **[OpenWeatherMap API](https://openweathermap.org/api)** (Core weather data & icons)

## 🚀 Installation & Local Setup

To get a local copy up and running, follow these simple steps:

1. Clone the repository:
   ```sh
   git clone https://github.com/nagarajpandith/weather-app
   ```
2. Navigate to the project directory:
   ```sh
   cd weather-app
   ```
3. Open `index.html` in your browser, or for the best development experience, use an extension like **Live Server** in VS Code.
